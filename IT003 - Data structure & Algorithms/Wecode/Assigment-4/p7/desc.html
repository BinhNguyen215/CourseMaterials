PROBLEM 7

Cài đặt cây nhị phân tìm kiếm (Binary Search Tree - BST), thực hiện thao tác thêm, duyệt và xóa.

INPUT:

Input trên mỗi dòng sẽ bao gồm 2 phần, n và x, nếu n = 1 thì thêm giá trị x vào cây, nếu n bằng 2 thì xóa node có giá trị là x trong cây, nếu n = -1 thì ngừng thao tác.

Khi xóa thất bại (không xóa được node nào do không có giá trị = x) thì cập nhật biến k++ (k là số lần xóa thất bại, khởi tạo k = 0).

OUTPUT

Xuất ra kết quả duyệt cây theo thứ tự Left Node Right (LNR) trên cùng một dòng.

Dòng thứ hai xuất ra k
